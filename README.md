## Convenciones
- Todos los nombres de archivos en minúsculas (solo los README en mayúsculas)
- Carpetas:
  - css: hojas de estilos
  - fonts: fuentes (tipografías)
  - img: imagenes
  - media: audio / video
  - templates: archivos html (excepto index.html)
